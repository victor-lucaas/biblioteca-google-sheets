B

Biblioteca de Informacoes

Um painel simples para criar acessos locais, conectar varias planilhas e consultar tudo pelo navegador.

Como comecar

1

Crie seu acesso

Informe nome, usuario e senha. Esse acesso fica salvo apenas neste navegador.

2

Conecte uma planilha

Cole a URL do App da Web do Google Apps Script. Cada URL pode representar uma biblioteca diferente.

3

Guarde informacoes

Salve titulo, categoria e conteudo. Cada registro vira uma linha na planilha selecionada.

4

Pesquise quando precisar

Escolha uma planilha, busque por texto ou filtre por categoria para encontrar rapidamente.

Entrar

Acesse suas planilhas cadastradas neste navegador.

Criar acesso

Depois de criar, voce podera cadastrar uma ou varias planilhas.

Aviso: este login e local, salvo no navegador. Nao use senha importante.

B

Biblioteca

Gerencie suas planilhas.

Opcoes

Guardar informacao

O registro sera enviado para a planilha ativa.

Selecionada Geral

Pegar informacoes

Consulte os dados salvos na planilha ativa.

Clique em pesquisar para carregar as informacoes.

Planilhas cadastradas

Voce pode ter varias bibliotecas, cada uma apontando para uma URL de App da Web.

Adicionar planilha

Cole a URL do App da Web. Ela deve terminar com /exec.

Gerenciar categorias

Crie categorias para organizar os registros desta conta.

!

Excluir informacao?

Essa acao remove o registro da planilha ativa e nao pode ser desfeita pelo site.
